# OpenClaw 16-MBTI スキル（人格化視点設定ツール）

## 📌 1. プロジェクト概要
これはOpenClawプラットフォーム向けに設計された「コミュニケーションスタイルおよび視点設定ツール」スキルです。
その核心的な目的はAIを占い師や性格テスターに変えることではなく、**MBTIの16種類の認知アーキテクチャをプラグ可能なシステムロールパッケージに変換すること**です。

このスキルを呼び出すことで、AIが瞬時に視点を切り替えられます。例えば：
- **INTJ（戦略的プランナー）** の冷徹な視点で冗長なビジネス計画書を精査し、論理的な欠陥を直接切り捨てる。
- **ESFJ（協調型推進者）** のトーンに切り替え、催促の目的を達成しながらも部門間の関係を壊さない柔らかなメールを作成する。
- または**デュアル比較モード**を直接起動して、極端な効率機械（ESTJ）と極端な共感の達人（INFP）が同じ再編成方案について左右から議論させる。

## 🚨 2. 強硬声明：境界と適用範囲の制限
Myers & Briggs Foundationおよび主流心理学界はすでに厳格なコンセンサスに達しているため、本スキルにはコードとロジックの根底に越えられない禁止ラインが組み込まれています：

1. **絶対【非診断ツール】**：本スキルは真の人間に対する「精神的健康判定」「抑うつ傾向評価」を拒否し、その能力もありません。
2. **絶対【非採用・解雇ツール】**：本スキルは「履歴書からINFPをすべてフィルタリングしてください」「このINTJの人物が経営幹部になれるか評価してください」などのリクエストを拒否します。MBTIが測定するのは認知の好みであり、スキル・IQ・業務パフォーマンスとは全く無関係です。人事選考への使用は重大な倫理的逸脱と潜在的な差別に関わります。

## ⚙️ 3. 対応する呼び出しモード

### 💎 モード1：指定ペルソナ単路出力（Single Persona）
ユーザーが特定のパーソナリティタイプと現在の問題の説明を明確に入力します。
- **例**：`「ESTPのスタイルで、だらだらと長い週報を書き直してください。」`
- **動作**：AIがESTPの言語辞書（短文・利益志向・行動力満載）を直接読み込み、長期的な情緒を無視して収益の核心に直接切り込む。

### 🤖 モード2：タスクマッチング自動推薦（Auto Recommend）
ユーザーが問題だけを投げかけ、システムが`task_mode`と16タイプの`適合タスク表`でマッチングを行います。
- **例**：`「チームの士気が低下し、みんなやる気がなく、内部の不満が大きいです。どうすればいいですか？」`
- **動作**：AIがこれを「感情的安定と集団的動員」タスクと識別し、すべてのT型（思考型）パイプラインを自動的に拒否し、**ENFJ（関係動員者）** または **ISFJ（細心の守護者）** を積極的に推薦し、理由を提示した上でどちらを採用するかをユーザーに確認する。

### ⚔️ モード3：マルチペルソナサンドボックス推演（Compare Perspectives）
ユーザーが2種類の全く異なるペルソナを指定し、認知の溝の衝突を表示させます。
- **例**：`「INTPとENTJの視点でそれぞれ私のシステムアーキテクチャ設計を評価してください。」`
- **動作**：AIがINTP（基盤理論の欠陥と更なる可能性に注目）とENTJ（いつ実装できるか・コストはいくらかに注目）の2つの極端な意見を同時にレンダリングし、最終的に「調停統合提案」を提示する。

## 📂 4. ディレクトリ構造ガイド
プログラムがシームレスに読み込めるよう、本プロジェクトはOpenClaw公式標準で構築されています：
- `SKILL.md`：能力の中枢神経。呼び出しロジックと出力フォーマット制約を定義。
- `profiles/`：データベースコア。16個の精細にチューニングされた独立したペルソナ設定ファイルを収録。
- `examples.md` & `tests.md`：呼び出し検証用のサンプルとサンドボックステスト基準。
- `03_personality_profiles.*`：他の自動化スクリプト向けの外部インデックスマップ。

## 🔧 5. パラメータ一覧（参考）
| 入力パラメータ | 型 | 必須 | 備考 |
| :--- | :--- | :--- | :--- |
| `personality_type` | 文字列 | 任意 | 16種類の標準コードのいずれか。省略するとRecommendモードに移行 |
| `task_mode` | 文字列 | 必須 | `analyze`、`create`、`review`、`comfort`、`organize` |
| `output_style` | 文字列 | 任意 | `single`、`recommend`、`compare` |
| `compare_types` | 配列 | 任意 | `[TypeA, TypeB]`、Compareモードのみ有効 |
